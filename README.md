# 📄 PDF to Structured Intelligence Tool

A web-based tool that extracts text and structure from PDF documents and converts them into clean, LLM-ready Markdown, stripped of unnecessary visual noise.

## 💡 Key Features
* **Intelligent Parsing:** Uses PyMuPDF4LLM to convert complex document content into structured Markdown.
* **Context Preservation:** Maintains the original layout, including tables and multi-column structures.
* **AI-Optimized:** Automatically strips out images, headers, and footers to provide a clean, high-signal text stream.
* **Direct Export:** Save your converted data instantly as an .md file.


## 🛠 Tech Stack
* **Language:** Python
* **Framework:** Streamlit
* **Parser:** [PyMuPDF4LLM](https://github.com/pymupdf/pymupdf4llm)

## 🚀 How to use
1. Upload your PDF file using the file uploader.
2. Click the "Convert" button.
3. Review the output directly in the app.
4. Click "Download Markdown" to save your file for your AI projects.
